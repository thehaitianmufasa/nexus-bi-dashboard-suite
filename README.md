# Nexus BI Dashboard Suite

**Enterprise Power BI Dashboard Suite — Manufacturing Operations**

A collection of five production dashboards built for plant leadership and cross-functional teams at a major manufacturing facility. Each dashboard replaced manual reporting processes with automated, real-time analytics.

## Dashboards

| Dashboard | Description |
|-----------|-------------|
| **Calibration Compliance** | Equipment calibration tracking across 514+ assets with compliance scoring, vendor breakdown, and certificate links |
| **Production Workforce** | OEE man-hour capacity planning with weekly trend analysis and 40hr/50hr target tracking |
| **MUV Analysis** | Material usage variance financial analysis tracking $6M+ in annual losses by reason code and production line |
| **Contractor Compliance** | Real-time contractor risk classification and insurance status tracking for 70+ vendors |
| **Contract Compliance Report** | Monthly contract lifecycle tracker covering 55+ vendor agreements by status and category |

## Tech Stack

- **Visualization:** Power BI, DAX measures, dynamic parameters
- **Data Modeling:** Star schema, Power Query (1,100+ row matching)
- **Infrastructure:** SharePoint cloud refresh, monthly file swaps
- **Source Systems:** SAP, Veriforce, Apttus, Excel
- **Portfolio Site:** HTML, CSS, JavaScript (zero dependencies)

## Features

- Interactive filtering, sorting, and search across all dashboards
- Multi-page navigation with tab and sidebar patterns
- Responsive layouts for desktop and tablet viewing
- KPI cards with real-time metric displays
- Color-coded status indicators and compliance tracking
- Pagination and view switching (table/card views)
